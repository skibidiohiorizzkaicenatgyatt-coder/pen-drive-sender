A este ponto você já deve ter o python (linguagem de programação) e git (sistema de controle de versão) instalados no PC
Por isso, falta baixar o pip (gerenciador de pacotes) e as bibliotecas necessárias

Siga os comandos

pip --version

Se não aparecer a versão dele, então ele não está instalado
Rode isso

python -m ensurepip --upgrade

Depois

pip --version

Se aparecer a versão, o pip está instalado

Agora você precisa baixar as bibliotecas, que no caso é apenas uma

Rode no isso terminal

pip install apscheduler

Agora o seu setup já deve estar pronto
